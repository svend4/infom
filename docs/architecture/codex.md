## IV. Три Геометрических Закона InfoM

Все три появились независимо в `infom`, `pro2`, `meta2` —
и оказались одним и тем же.

### Закон Q6: Любую сущность можно адресовать 6 битами

```
Сущность X → embed(X) → average_pool → Q6 hex-адрес [0..63]
```
Это не упрощение — это сжатие без потери топологии.
64 ячейки достаточно для навигации в семантическом пространстве.

### Закон Тангрэма: Форма = набор геометрических примитивов

```
Сообщество / Фрагмент / Тип → convex_hull → SimplifiedPolygon → TangramSignature
```
Две сущности с похожей формой — вероятно, семантически близки.

### Закон Фрактала: Граница = мера сложности

```
Граница объекта → box_counting → FractalDimension ∈ [1.0, 2.0]
```
Простые концепции имеют гладкую границу (D≈1).
Сложные, многогранные — фрактальную (D→2).
